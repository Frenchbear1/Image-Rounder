# Image Rounder

An offline browser tool for cropping images, rounding corners, and exporting polished image files.

## Capabilities
- Tap the empty editor or drop an image to upload locally (no server required)
- Drag crop edges, corners, and the selected crop area directly on the image
- Pull the blue corner handle to adjust the radius without sliders
- Clear the current image without refreshing
- Lock common aspect ratios or crop freely
- Preview on dark, light, or grid backgrounds
- Export the processed image as PNG, JPG, or WebP
- Single-file static site
